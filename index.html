1. html : The root element of an HTML document. It contains all other HTML elements.

2. head : Contains meta-information about the document, such as its title and links to scripts or stylesheets.

3. title : Specifies the title of the document, which is displayed in the browser's title bar or tab.

4. img : Incorporate in-line graphics (typically icons or small graphics) into an HTML document.

5. style : Contains CSS rules for styling HTML content. It's used here to set visual styles for various elements in the document.

6. div : A container tag that is used to group together and apply styles to other elements.

7. body : Contains the content of the document. Everything that is displayed in the browser window, apart from the title, is in the body.

8. p : Represents a paragraph of text.

9. h1 : Represents a level 1 heading. There are six levels of headings in HTML, from h1 (most important) to h6 (least important).

10. h2 : Represents a level 2 heading

11. h3 : Represents a level 3 heading

12. b : Makes text bold. Note that for semantic purposes, it's often better to use the strong tag when you want to emphasize importance, and b for stylistic purposes without conveying extra importance.

13. br : Inserts a line break. Used for creating breaks inside elements like p , where whitespace doesn't create breaks.

14. style: An attribute that can be used on most HTML elements to apply inline styles. It's generally a better practice to use external stylesheets or the style tag in the head for bigger styling tasks, but for quick, element-specific styles, this attribute is useful.

15. class: An attribute that specifies one or more class names for an element. The class attribute is mostly used to point to a class in a style sheet.

16. footer: A structural element used to identify the footer of a page, document, article, or section.

17. font choice: Kode Mono - A custom-designed typeface explicitly created for the developer community.
